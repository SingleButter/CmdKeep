# Command Maid - 快速开始指南

## 第一步：安装必需软件

### 1. 安装 Node.js
- 访问: https://nodejs.org/
- 下载并安装 LTS 版本
- 安装完成后，在命令行运行 `node --version` 验证

### 2. 安装 Rust
- 访问: https://www.rust-lang.org/tools/install
- Windows 用户: 先安装 Microsoft C++ Build Tools
  - 下载地址: https://visualstudio.microsoft.com/visual-cpp-build-tools/
  - 安装时选择 "Desktop development with C++"
- 然后安装 Rust (使用 rustup)
- 安装完成后，运行 `rustc --version` 验证

## 第二步：运行应用

### 方式 1: 开发模式运行

1. 打开命令行，进入项目目录:
```bash
cd C:\develop\ClaudeCode\CommandMaid\command-maid
```

2. 安装 npm 依赖:
```bash
npm install
```

3. 运行开发模式:
```bash
npm run tauri dev
```

首次运行会编译 Rust 代码，可能需要几分钟时间。之后启动会快很多。

### 方式 2: 构建可执行文件

1. 构建应用:
```bash
npm run tauri build
```

2. 构建完成后，可执行文件位于:
```
src-tauri\target\release\command-maid.exe
```

双击运行即可。

## 常见问题

### Q: 运行时提示找不到 Rust 编译器
A: 确保已安装 Rust，并且重启了命令行窗口（环境变量需要刷新）

### Q: 编译时出现 C++ 相关错误
A: 确保已安装 Microsoft C++ Build Tools

### Q: npm install 很慢或失败
A: 可以尝试使用国内镜像:
```bash
npm config set registry https://registry.npmmirror.com
```

### Q: 找不到数据文件
A: 数据库文件自动保存在:
- Windows: `%APPDATA%\com.commandmaid.app\commands.db`

## 使用技巧

1. **快速添加命令**: 点击右上角"添加命令"按钮
2. **搜索**: 在顶部搜索框输入关键词即可实时搜索
3. **分类**: 左侧可以按分类筛选命令
4. **复制**: 点击命令卡片上的"复制"按钮，一键复制到剪贴板
